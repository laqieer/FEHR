Fire Emblem Heroes Remake on Game Boy Advance.

![title screen](res/gfx/raw/title_screen.png)

A hack based on Fire Emblem 7 Japanese version.

## Dependancy
* devkitpro
* cmake(>=3.13)

## Build
### Build Instruction
```
cp <game_rom> rom/fe7-jp.gba
./configure
cd build && make
```
